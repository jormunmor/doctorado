Symbolic Planner section

In this section you can review and download all the files related to the Symbolic Planner. The available material includes the Domain File, the Problem File as well as the Solution File. The Domain and Problem files are in the JShop2 format, which is very similar to a PDDL description. The Solution File contains all the primitives that conform the final plan.
